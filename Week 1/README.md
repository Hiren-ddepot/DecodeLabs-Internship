# Week 1 – Data Cleaning & Preparation

## Project Objective

The objective of this project was to clean and prepare a raw e-commerce dataset by identifying missing values, checking for duplicate records, and validating data formats.

## Tasks Performed

### Missing Values

* Identified missing values in the CouponCode column.
* Replaced 309 blank entries with "No Coupon".

### Duplicate Check

* Checked the dataset for duplicate records.
* Verified that no duplicate Order IDs existed.

### Data Validation

* Confirmed that all dates were correctly formatted.
* Validated numeric fields such as Quantity, UnitPrice, ItemsInCart, and TotalPrice.
* Ensured text fields were consistent and complete.

## Outcome

The dataset was successfully cleaned and prepared for analysis. All missing values were addressed, duplicate checks were completed, and data formats were validated to ensure accuracy and reliability.
